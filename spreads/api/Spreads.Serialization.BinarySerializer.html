SizeOf<T>(T, out BufferWriter, SerializationFormat)

Returns the size of serialized value payload plus Size plus PayloadLengthSize if payload is not fixed size. You could use the returned value to allocate a buffer for subsequent write operation - it must have capacity greater or equal to the return value of SizeOf<T>(T, out BufferWriter, SerializationFormat).

When serialized payload length is only known after serialization, which is the case for variable size types and many containers, this method serializes the value into payload. The buffer payload.temporaryBuffer.temporaryBuffer is owned by the caller, ownership is transferred to a Write method that disposes the buffer.

Returned payload.actualFormat.actualFormat could differ from preferredFormat when preferred format is binary but there is no custom binary serializer or if preferred format is compressed but payload length is less than CompressionStartFrom.

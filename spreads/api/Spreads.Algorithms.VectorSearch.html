Class VectorSearch

Algorithms to find values in contiguous data (memory region or a data structure with an indexer), e.g. Vec<T>. WARNING: Methods in this static class do not perform bound checks and are intended to be used as building blocks in other parts that calculate bounds correctly and do performs required checks on external input.

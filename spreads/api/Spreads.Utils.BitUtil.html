Align(Int32, Int32)

Align a value to the next multiple up of alignment. If the value equals an alignment multiple then it is returned unchanged.

This method executes without branching. This code is designed to be use in the fast path and should not be used with negative numbers. Negative numbers will result in undefined behavior.

FindNextPositivePowerOfTwo(Int32)

Fast method of finding the next power of 2 greater than or equal to the supplied value.

If the value is <= 0 then 1 will be returned.

This method is not suitable for MinValue or numbers greater than 2^30.

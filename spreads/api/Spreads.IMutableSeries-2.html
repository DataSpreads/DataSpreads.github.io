TryRemoveMany(TKey, TValue, Lookup)

Update value at key and remove other elements according to direction. Value of updatedAtKey could be invalid (e.g. null for reference types) and ignored, i.e. there is no guarantee that the given key is present after calling this method. This method is only used for atomic RemoveMany operation in SCM and could be not implemented/supported.

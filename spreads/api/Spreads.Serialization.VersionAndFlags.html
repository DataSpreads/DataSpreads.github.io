Remarks

Format: 0 1 2 3 4 5 6 7 8 +-+-+-+-+-+-+-+-+ |0|CNV|R|T|CMP|B| +---------------+ B - Binary format (read as "Not JSON"). If not set then the payload is JSON, if set then payload is blittable or custom binary. CMP - compression method (Zstd/Lz4/Deflate/None): 00 - not compressed 01 - GZip 10 - Lz4 11 - Zstd T - Timestamped. A value has Timestamp (8 bytes) as the first element of payload. It is included in payload length for varsized types. R - reserved. CNV - Converter version. 0 - will need completely new layout when this is not zero.

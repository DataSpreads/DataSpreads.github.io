TryFindAt(TKey, Lookup, out KeyValuePair<TKey, TValue>)

The method finds value according to direction, returns false if it could not find such a value. For indexed series LE/GE directions are invalid (throws InvalidOperationException), while LT/GT search is done by index rather than by key and is possible only when a key exists. TryFindAt works only with existing keys and is a series counterpat of MoveAt(TKey, Lookup).

FixedSize

Returns a positive size of a pinnable type T, -1 if the type T is not pinnable or has a registered IBinaryConverter<T> converter. We assume the type T is pinnable if GCHandle.Alloc(T[2], GCHandleType.Pinned) = true. This is more relaxed than Marshal.SizeOf, but still doesn't cover cases such as an array of KVP[DateTime,double], which has contiguous layout in memory.

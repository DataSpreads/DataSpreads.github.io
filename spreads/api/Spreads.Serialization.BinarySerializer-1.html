SizeOf(T, BufferWriter)

Returns the size of serialized value payload. When payload parameter is not null this method should additionally serialize the value into payload BufferWriter.

Parameters
Type Name Description

Value to serialize.

If not FixedSize then this parameter is not null and serializer must write actual payload into it.
Returns
Type Description
Int32
Remarks

For variable size types performing actual serialization to a temp buffer is often the only way to calculate the payload size. Write to payload directly in that case and do not use a separate buffer. The payload buffer content will then be used by BinarySerializer Write methods instead of performing serialization the second time.

Do not include length prefix in payload because BinarySerializer does so automatically and guarantees than Read(DirectBuffer, out T) method will receive the source buffer of exact payload length.

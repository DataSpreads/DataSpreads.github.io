Vec(Void*, Int32)

Creates a new Vec<T> over the target unmanaged buffer. This is quite dangerous, because we are creating arbitrarily typed T's out of a void*-typed block of memory. And the length is not checked. But if this creation is correct, then all subsequent uses are correct.
